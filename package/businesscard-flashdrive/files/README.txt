It is based on "Business Card Linux" by George Hilliard
https://github.com/thirtythreeforty/businesscard-linux

This edition was developed by Sol Huebner
(https://www.linkedin.com/in/solhuebner)

The user name is root and you should be able to guess the password :)

To connect to it please use a serial connection with the following options:

Baud rate: 115200
Data bits: 8
Stop bits: 1
Parity: None
Flow control: XON/XOFF

For your convinience, the provided putty.exe could be used
either manually (see below) or started via "ConnectViaPutty.bat":

"Serial line": COM1
(depending on your system the number might need to be higher)

"Speed": 115200
"Connection type": Serial

If you want to save these settings type in a name in "Saved Sessions"
and click "Save". The next time you use Putty you can just double click
on the name you chose to open the session.
